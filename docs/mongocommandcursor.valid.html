(PECL mongo >=1.5.0)

MongoCommandCursor::valid — Checks if the cursor is reading a valid result

Description

public MongoCommandCursor::valid(): bool

Parameters

This function has no parameters.

Return Values

true if the current result is not null, and false otherwise.
