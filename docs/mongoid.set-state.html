MongoId::__set_state

(PECL mongo >= 1.0.8)

MongoId::__set_state — Create a dummy MongoId

This extension that defines this method is deprecated. Instead, the MongoDB extension should be used. There is no equivalent for this method in the new extension.

Description

public static MongoId::__set_state(array $props): MongoId

This function is only used by PHP internally, it shouldn't need to ever be called by the user.

It is identical to the function:

<?php

public static function __set_state($props) {
    return new MongoId("000000000000000000000000");
}

?>

Parameters

props

Theoretically, an array of properties used to create the new id. However, as MongoId instances have no properties, this is not used.

Return Values

A new id with the value "000000000000000000000000".
